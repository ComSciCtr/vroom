random_vertex

utils/generators.py

tags:

  • utils
  • generators

Generate a random vertex.

syntax

random_vertex()

optional arguments

start minimum for vertex data
stop maximum for vertex data

return

Random vertex with x,y,z values in the range ['start', 'stop']

examples

randomly placed cubes

def display():
   lighting(False)
   for i in range(10):
      pushMatrix()
      translate(random_vertex(-5, 5))
      cube(3.0)
      popMatrix()
